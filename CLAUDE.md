## Design Context

This project has a full design system. Before writing any UI code or making visual decisions, read:

- **PRODUCT.md** — register (product), users, brand personality (Trustworthy, Premium, Human), anti-references, and 5 design principles
- **DESIGN.md** — color tokens (OKLCH), typography (Lexend headings / Source Sans 3 body), elevation rules, component specs, and Do's/Don'ts

Creative North Star: **"The Trusted Introducer"** — premium but personal, human-verified, calm authority. Never startup-bro, never LinkedIn clone, never legacy government dashboard.
